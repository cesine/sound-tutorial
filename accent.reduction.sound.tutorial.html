Instalation:

Download Audacity, a free open source sound editor.

Windows XP: audacity-win-1.2.3.exe
For other types of Operating systems see the Audacity website
http://audacity.sourceforge.net

Install

Open:

Save the file tunisian.soapopra5left.5.mp3 in the My Documents Folder
Open Audacity (you can probably find it in the Programs section of the Start menu.
Select File >> Open... and find tunisian.soapopra5left.5.mp3

The wave form of the file will display in a window. Click on the green triangle to listen to the file (you can also hit the Space bar).
This is an extract from a popluar soap opera in Tunisia. The soap opera is originally from Mexico. How can you tell?
Can you guess what the speaker is saying? What do you hear?

View the Intonation using Pitch:

In the top left corner of the waveform window you can see a title that says "tunisian.soa" and a down arrow. Click on the down arrow, select "Pitch (EAC)"

Play the file again. Notice the pink and white spots. These indicate the "pitch" of hte voice. Our vocal cords (actually folds of skin) vibrate like a guitar string. But unlike a guitar string our vocal folds can become tighter and looser to make different pitches. We use this when we speak. The pink and white areas indicate which pitches are strongest. Try to imagine what she would be saying in English. You can guess quite acqurately without even knowing the words or the individual sounds shes making.

Save the file roni.intonation.mp3 in the My Documents Folder
Open the file useing Audacity.

This is a recording of my little sister, Roni, when she was 5. I asked her to tell me a story. Listen to her version of a story in 15 seconds ;) Notice that her pitch falls when she finishes a thought group.

Save the file meera.phone.call031213.04.5.mp3 in the My Documents Folder
Open the file using Audacity.

This is a recording of my old roomate while she was talking to a friend on the phone in Mauritian French Creole. Listen to the file and look at the pitch. What do you find different and the same between Meera and the other speakers?

View the Segments using Spectrum:

Save the file speak.easy.passage_1.mp3 in the My Documents Folder
Open the file with audacity
Click on the title of the sound wave window. Select "Spectrum."

Record:
Close the sound wave windows. Click on the red Record button. Check to see if your mic is connected.
Save a File:
Go to File >> Export as MP3
Audacity will ask you to locate the lame_enc.dll this is the library for creating mp3s. Dont close the window. Do a search on you computer for the file or download it from http://cesine.ca/downloads/ save it somewhere you can find it. Then locate the file in the window Audacity provided.
